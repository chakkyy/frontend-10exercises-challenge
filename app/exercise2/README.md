**Scenario:**  
You're asked to improve a form for newsletter signup:  
```html
<form>
  <input type="text" placeholder="Your email" />
  <button>Submit</button>
</form>
```  
**Task:**  
- Rewrite this form to:  
  1. Be semantic and accessible.  
  2. Support basic HTML5 validation.
